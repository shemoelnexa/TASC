# TASC Outsourcing — Homepage Redesign

Homepage redesign concept for TASC Outsourcing, the Middle East's leading recruitment and HR solutions provider.

## Tech Stack
- Pure HTML/CSS (single file, images base64-embedded)
- GSAP 3.12 + ScrollTrigger (scroll animations)
- Lenis (smooth scroll)
- Vercel (deployment)

## Local Development
Open `index.html` in a browser. For the hero video to load, serve via a local server:
```bash
npx serve .
```

## Deploy to Vercel
1. Push to GitHub
2. Import repo in [vercel.com/new](https://vercel.com/new)
3. Deploy — no build config needed
